# Real Estate Platform 

##  Overview
Real Estate Platform is a full-stack web application built with Django following the MVC architecture. The system simulates a real-world real estate company where users can browse properties, apply advanced filters, estimate property prices, and calculate mortgage loans online.

The platform is designed for a single real estate company and includes role-based functionality for users and agents, along with a management system for property listings and user interactions.

---

## Key Features

### Property Search & Filtering
- Search properties using keywords
- Advanced filtering system including:
  - Price range
  - Location
  - Property type
  - Agent-based filtering
- Fast and user-friendly browsing experience

---

### Property Price Estimation
- Users can estimate the approximate market value of a property
- Based on property specifications such as size, location, and type
- Provides quick insights into potential listing prices

---

### Mortgage / Loan Calculator
- Users can calculate estimated mortgage payments online
- Includes:
  - Interest rate calculations
  - Monthly installment estimation
  - Loan duration impact
  - Total repayment overview
- Helps users understand financing options before purchasing

---

### Agent Dashboard (Single Company System)
- Each agent has a personal dashboard
- Agents can:
  - Add new property listings
  - Edit and delete properties
  - Manage their own listings
- Centralized system for a single real estate company

---

### Feedback System
- Users can send feedback or inquiries
- Enables communication between users and the platform/company

---

## Technologies Used
- **Backend:** Django (Python)
- **Frontend:** HTML, CSS, JavaScript
- **Styling:** Bootstrap
- **Database:** SQLite (default Django database)
