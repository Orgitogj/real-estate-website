# Car Rental Platform

## Overview
Car Rental Platform is a full-stack web application built with Django that simulates a real-world car rental system. The platform allows multiple companies to list and manage their rental vehicles, while users can search, filter, and book cars based on real-time availability.

The system is designed with role-based access control, supporting three main roles: customers, companies, and administrators.

---

## Key Features

### Customer Functionality
- User registration and authentication
- Personal profile management
- Browse available cars
- Advanced search and filtering
- View real-time car availability
- Online booking and reservation system

### Company Functionality
- Dedicated company dashboard
- Add, update, and delete car listings
- Manage vehicle availability
- Control over company-specific data

###  Admin Functionality
- Full access via Django Admin panel
- Manage users, companies, and listings
- Monitor system activity

---

## Technologies Used
- **Backend:** Django (Python)
- **Frontend:** HTML, CSS, JavaScript
- **Styling:** Bootstrap
- **Database:** MySQL
